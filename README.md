# PHP MySQL CRUD Application

This is a simple CRUD (Create, Read, Update, Delete) application built using PHP and MySQL.

## Features
- Add post with image
- View posts
- Edit post
- Delete post
- Image upload support

## Technologies Used
- PHP
- MySQL
- XAMPP
- HTML

## How to Run
1. Install XAMPP
2. Copy project folder to `htdocs`
3. Create database in phpMyAdmin
4. Import table
5. Run `http://localhost/crud`
